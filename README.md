## Installation

You can install nanoChain directly from PyPI using pip:

```bash
pip install nanoChain
```

Alternatively, you can clone the repository and install from source:

```bash
git clone https://github.com/jamesliu/nanoChain.git
cd nanoChain
pip install .
```
